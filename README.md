# Installation
0. Open VS Code, make sure you have these extensions installed: Jupyter, Python.

1. Make a new terminal, make sure it's on the root directory of the project and Create python venv  
`python -m venv venv`

2. Activate python venv  
On Windows: `venv\Scripts\activate`  
On Linux/MacOS: `source venv/bin/activate`

3. Install dependencies  
`pip install -r requirements.txt`

4. On VSCode, select python version on the right side, and select the venv one. It's usually have `venv/bin/python` path for Linux/MacOS.

5. Jupyter is ready to run
